# zozincanvas

이미지 대충 올리고, 필요한 부분만 빠르게 가리고 표시하고 저장하는 정적 이미지 편집기다.

서버에 이미지를 올리지 않는다. 브라우저 안에서만 처리한다. 그래서 개인정보 들어간 캡처, 얼굴 나온 사진, 블로그에 올릴 이미지 편집하기 좋다.

## 구현 방식

이미지는 `File API`로 읽고 `Canvas`에 그린다.

펜이랑 블러는 픽셀에 바로 반영한다. 이건 브러시 도구라서 객체로 들고 있을 이유가 별로 없다.

도형, 스티커, 텍스트는 객체로 따로 들고 있다. 그래서 나중에 선택해서 이동하거나 크기를 바꿀 수 있다.

대략 구조는 이렇다.

```text
baseCanvas: 업로드 이미지 + 펜 + 블러
objects: 도형, 스티커, 텍스트
editorCanvas: baseCanvas와 objects를 합쳐서 보여주는 화면
```

다운로드할 때는 현재 보이는 결과를 다시 캔버스에 합쳐서 이미지로 저장한다.

텍스트는 캔버스 자체에서 직접 입력할 수 없어서, 선택한 텍스트 박스 위치 위에 `textarea`를 잠깐 띄운다. 입력이 끝나면 텍스트 객체에 반영하고 다시 캔버스에 그린다.

스포이드는 브라우저가 `EyeDropper API`를 지원하면 그걸 먼저 쓴다. 안 되면 캔버스를 클릭해서 해당 픽셀 색상을 읽는다.

## 사용 방법

1. 이미지 올린다.
   - `이미지 업로드` 박스를 클릭해도 되고, 파일을 끌어다 놓아도 된다.
   - 편집 영역에 바로 드롭해도 된다.

2. 가릴 건 블러로 문지른다.
   - `블러` 선택
   - 브러시 크기와 블러 강도 조절
   - 얼굴, 이름, 전화번호 같은 부분을 문지르면 된다.

3. 표시할 건 펜이나 도형으로 표시한다.
   - 펜은 그냥 드래그
   - 도형은 드래그해서 크기 잡기
   - 화살표도 드래그로 방향 잡기

4. 텍스트는 편집창에서 바로 쓴다.
   - `텍스트` 선택
   - 캔버스에서 텍스트 박스 크기만큼 드래그
   - 그 자리에서 바로 입력
   - 기존 텍스트는 `선택` 도구로 더블클릭해서 다시 편집

5. 이미 만든 도형, 스티커, 텍스트는 `선택`으로 고친다.
   - 클릭해서 선택
   - 드래그해서 이동
   - 우하단 핸들을 드래그해서 크기 조절

6. 색상은 세 가지 방식으로 고른다.
   - 기본 color picker
   - 프리셋 스와치
   - 헥사코드 입력
   - 스포이드로 화면/캔버스 색 찍기

7. 저장한다.
   - 파일명 입력
   - `PNG`, `JPG`, `WEBP` 중 선택
   - 다운로드

## 로컬 테스트

그냥 정적 파일이라 서버 없이 열어도 어느 정도 된다.

그래도 브라우저 보안이나 파일 경로 이슈 피하려면 로컬 서버로 보는 게 낫다.

```powershell
cd C:\zozindev\zozincanvas
C:\Users\조지\.cache\codex-runtimes\codex-primary-runtime\dependencies\python\python.exe -m http.server 4173 --bind 127.0.0.1
```

브라우저에서 연다.

```text
http://127.0.0.1:4173/
```

간단 검증은 이 정도 보면 된다.

- 이미지 업로드 되는지
- 드래그 앤 드롭 되는지
- 펜/블러가 모바일 터치에서도 먹는지
- 도형/스티커/텍스트 선택과 크기 조절 되는지
- 텍스트 직접 편집 되는지
- 스포이드 색상 추출 되는지
- `PNG`, `JPG`, `WEBP` 다운로드 되는지

## 운영 방법

운영은 단순하게 가면 된다.

- 기능 추가하면 로컬에서 먼저 직접 만져본다.
- `node --check script.js`로 문법 확인한다.
- 브라우저에서 업로드, 편집, 다운로드를 직접 확인한다.
- 문제 없으면 커밋하고 푸시한다.
- Cloudflare Pages가 GitHub main 브랜치를 보고 있으면 자동 배포된다.

지금 구조에서는 서버 로그나 DB 백업 같은 운영 포인트가 없다. 대신 브라우저 호환성이 중요하다. 특히 `EyeDropper API`는 브라우저마다 다를 수 있어서 fallback 캔버스 스포이드를 같이 둔 상태다.
